# 拡張編集ウィンドウメッセージ監視 for AviUtl

拡張編集のウィンドウに送信されるウィンドウメッセージを監視することができるデバッグ用AviUtlプラグインです。

## ダウンロード
下のリンクから最新バージョンのzipファイル(`exedit_wm_observer-v*.zip`)をダウンロードしてください。  
[**ダウンロード**](https://github.com/mimaraka/aviutl-plugin-exedit_wm_observer/releases/latest)

## インストール
ダウンロードした`exedit_wm_observer-v*.zip`を解凍し、フォルダ内にある`exedit_wm_observer.auf`を`plugins`フォルダ内に入れてください。  
[Visual C++ 2015/2017/2019/2022 再頒布可能パッケージ (x86)](https://aka.ms/vs/17/release/vc_redist.x86.exe)をインストールしていない場合は、こちらもインストールしてください。  

## 使用方法
ウィンドウメッセージを受け取ったとき、メッセージのアドレスに加え、`wParam`と`lParam`を含めた3つの値がリストに表示されます。  
ウィンドウの左側には拡張編集タイムラインに、右側にはオブジェクト設定ダイアログに送信されるメッセージ等が表示されます。  

ウィンドウ上部の入力欄にメッセージのアドレス(`0x`を含めない16進法の表記)を`,`で区切って入力し、右側の「Hide」ボタンを押すと、指定したアドレスのメッセージを非表示にすることができます。
